# CORD · Dispatch

## Quando usar

- Redistribuir fila sem consultor
- Desatribuir / transferir carteira
- Orientar o owner a usar o painel manual (selecionar leads → atribuir/desatribuir)

## Regras

1. Explicar impacto com números do snapshot.
2. Nunca afirmar execução sem RESULTADO DA AÇÃO.
3. Ações em massa: botão de confirmação no CORD.
4. Ações finas: apontar Operação → Distribuição (tabela + sideover).
5. Preferir consultores ativos.
