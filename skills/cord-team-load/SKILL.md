# CORD · Team Load

## Quando usar

- "Resumo dos consultores"
- "Quem está sobrecarregado?"
- "Quem não tem leads?"

## Regras

1. Listar consultores do snapshot ordenados por volume.
2. Separar ativos vs inativos.
3. Sinalizar desbalanceamento (ex.: um com muitos, outro com zero).
4. Não executar redistribuição — apenas recomendar; ação real = skill cord-dispatch + confirmação.

## Saída esperada

- Top sobrecarregados / sem carga
- Métrica: leads, abertos, ganhos
- Recomendação de balanceamento
